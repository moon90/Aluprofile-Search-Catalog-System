Client Intake Package - Oliver

How to use:
1. Fill each CSV file in Excel (one row per item).
2. Keep ID fields stable (do not change once used).
3. Put media files into the `uploads/` folders using the naming rules below.
4. Return the full `client-intake-template` folder.

CSV files:
- `profiles_master.csv`
- `suppliers_master.csv`
- `applications_master.csv`
- `cross_sections_master.csv`
- `admin_users.csv`
- `acceptance_checklist.csv`

File naming rules:
- Drawings: `PROFILECODE_drawing_v1.pdf` or `PROFILECODE_drawing.png`
- Profile photos: `PROFILECODE_photo_01.jpg`
- Supplier logos: `SUPPLIERCODE_logo.png`

Folder structure:
- `uploads/01_profiles_drawings/`
- `uploads/02_profiles_photos/`
- `uploads/03_supplier_logos/`
- `uploads/04_branding/`
- `uploads/05_optional_docs/`

Branding folder content:
- Main logo (SVG/PNG)
- Color palette (text or PDF)
- Preferred fonts (if any)

Optional docs:
- Existing catalogs, scans, reference screenshots, notes.
